# IoTSim: Virtual Sensor Network Simulator

A digital twin IoT simulator that generates virtual sensor data streams for testing IoT systems without physical sensors.

## Project Overview

This system simulates various sensors including:
- **Temperature sensors** (18-25°C with daily variations)
- **Humidity sensors** (30-70% with realistic patterns)  
- **Water flow sensors** (0-100 L/min)

The architecture follows a simple three-component pipeline:
Sensor Simulator (Python) → CSV File Storage → Dashboard (Streamlit)
