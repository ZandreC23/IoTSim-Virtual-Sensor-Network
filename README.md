# IoTSim: Virtual Sensor Network Simulator

A digital twin IoT simulator that generates virtual sensor data streams for testing IoT systems without physical sensors.

## Project Overview

This system simulates various sensors including:
- Water sensors
- Temperature sensors  
- Humidity sensors

The architecture follows an end-to-end pipeline:
**Sensor Simulator → Data Stream → Processing Service → Dashboard**
